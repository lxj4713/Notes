内存管理

python 中内部管理机制：

gc机制 garbage collection （垃圾回收机制）

1. 引用计数机制
``` python
import gc
sys.getrefcount()  #获取引用计数

class Demo(object):
    def __init__(self):
        print(id(self))

    def __new__(cls, *args, **kwargs):
        print('NEW')
        return super(Demo,cls).__new__(cls)

    def __del__(self):
        print('over')

s = Demo()
print('程序结束')
'''
NEW
140636110471760
程序结束
over
'''

s = Demo()
del s
print('程序结束')
'''
NEW
140034811843152
over
程序结束
'''

#创建了对象s，初始计数为2，当代码运行结束后，会自动销毁对象s，释放内存
#当主动销毁对象释放内存的话，代码运行结束后不会最后再次销毁对象
```
